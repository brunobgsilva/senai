CSS

O que é CSS?




O CSS (Folha de Estilo em Cascatas) serve para estilizar documentos HTML, definindo como os elementos visuais devem ser exibidos em uma página da web. Ele permite separar a estrutura (HTML) do estilo (CSS), facilitando a manutenção e a atualização do visual de um site.



Em outras palavras, o CSS é responsável pela aparência de uma página web, controlando cores, fontes, layout, espaçamento e outros elementos visuais.
