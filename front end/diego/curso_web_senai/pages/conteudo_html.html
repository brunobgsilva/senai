HTML - HyperText Markup Language

HTML é a linguagem usada para estruturar páginas web. Abaixo estão as principais tags organizadas por função:

1. Estrutura básica
  • <!DOCTYPE html> – Declara o tipo do documento
  • <html> – Elemento raiz
  • <head> – Cabeçalho com metadados
  • <body> – Conteúdo visível da página

2. Títulos e parágrafos

  • <h1> a <h6> – Títulos de diferentes níveis
  • <p> – Parágrafo de texto

3. Listas

  • <ul> – Lista não ordenada
  • <ol> – Lista ordenada
  • <li> – Item de lista

4. Links e imagens

  • <a href="url"> – Link para outra página
  • <img src="imagem.jpg" alt="Descrição"> – Imagem

5. Tabelas

  • <table> – Tabela
  • <tr> – Linha da tabela
  • <td> – Célula de dados
  • <th> – Célula de cabeçalho

6. Formulários

  • <form> – Formulário
  • <input>, <textarea>, <select>, <button> – Campos do formulário
  • <label> – Texto associado a campos

7. Semântica e layout

  • <header>, <footer>, <nav>, <main>
  • <section>, <article>, <aside>, <div>

8. Texto e formatação

  • <strong>, <em> – Destaque (negrito e itálico)
  • <br> – Quebra de linha
  • <hr> – Linha horizontal
  • <code> – Bloco de código
  • <span> – Texto inline
